# 🚀 Schnellstart – KI-Mitarbeiter Kontrollzentrum

## Einmalige Einrichtung (5 Minuten)

```bash
# 1. Repo herunterladen
git clone https://github.com/jasmindipardo/ki-mitarbeiter-kontrollzentrum.git
cd ki-mitarbeiter-kontrollzentrum/dashboard

# 2. Starten (macht alles automatisch)
bash start.sh
```

Das Start-Script:
- installiert alle Abhängigkeiten automatisch
- fragt einmalig nach einem Passwort
- zeigt dir den Link zum Dashboard
- öffnet den Browser automatisch (wenn möglich)

---

## Danach: Täglich starten

```bash
bash start.sh
```

Im Terminal erscheint:
```
══════════════════════════════════════
  ✅ Dashboard läuft!
══════════════════════════════════════

  🔗 http://100.xx.xx.xx:7433

  👆 Diesen Link im Browser öffnen
══════════════════════════════════════
```

Diesen Link bookmarken → beim nächsten Mal einfach im Browser öffnen!

---

## Autostart (nie mehr Terminal!)

Einmalig einrichten, dann startet das Dashboard automatisch beim Server-Start:

```bash
sudo bash setup-autostart.sh
```

Danach nie mehr manuell starten. Dashboard läuft immer.

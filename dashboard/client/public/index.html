So liest du einen eigenen Zeitplan — 5 Stellen, immer:
0 8 * * 1 ↑ ↑ ↑ ↑ ↑ │ │ │ │ └─ Wochentag: 1=Mo 2=Di 3=Mi 4=Do 5=Fr 6=Sa 0=So │ │ │ └────── Monat (1–12 oder * = jeden Monat) │ │ └─────────── Tag (1–31 oder * = jeden Tag) │ └──────────────── Uhrzeit (0–23) └───────────────────── Minute (immer 0 = volle Stunde)
* bedeutet "immer/egal"  ·  */3 bedeutet "alle 3" (z.B. alle 3 Stunden)
👉 tägl. 17 Uhr: 0 17 * * *  |  Mo. 9 Uhr: 0 9 * * 1  |  1. des Monats: 0 1 1 * *
Tipp: Einfach oben klicken — Code wird automatisch eingetragen.
💡 Wann ist "alle 10 Sekunden" sinnvoll? Nur für sehr zeitkritische Aufgaben wie Live-Monitoring oder schnelle Reaktionen auf eingehende Daten. Für normale Aufgaben (Berichte, Erinnerungen, Checks) lieber "alle 30 Min" oder stündlich — spart API-Kosten.

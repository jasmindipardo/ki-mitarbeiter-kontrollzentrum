## Terminal-Befehle (Mac/Linux)

```bash
# Status anzeigen
tailscale status

# Verbinden
sudo tailscale up

# Trennen
sudo tailscale down

# IP anzeigen
tailscale ip

# Version anzeigen
tailscale version
```

---

## Windows

```powershell
# In der PowerShell oder cmd:
tailscale status
tailscale ip
```

---

## Was bedeutet meine IP?

Deine Tailscale-IP beginnt immer mit `100.` – zum Beispiel `100.94.12.45`.  
Diese IP gehört nur dir und bleibt für immer gleich.

---

## Gerät offline?

1. Tailscale auf diesem Gerät öffnen
2. Sicherstellen, dass du eingeloggt bist
3. "Connect" klicken (falls nicht aktiv)

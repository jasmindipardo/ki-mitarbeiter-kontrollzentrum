# Tailscale einrichten – Windows

## Schritt 1: Herunterladen
1. Gehe zu [tailscale.com/download](https://tailscale.com/download)
2. Klicke auf **Download for Windows**
3. Führe die `.exe`-Datei aus

## Schritt 2: Installieren
1. Dem Installationsassistenten folgen
2. Auf **Install** klicken
3. Windows fragt nach Administratorrechten → **Ja**

## Schritt 3: Einloggen
1. Tailscale öffnet sich automatisch in der Taskleiste (unten rechts)
2. Klicke auf das Tailscale-Symbol
3. Klicke auf **Log in**
4. Browser öffnet sich → mit deinem Account einloggen (Google, GitHub oder E-Mail)

## Schritt 4: Fertig!
- Dein Gerät erscheint jetzt im Tailscale-Dashboard
- Du siehst eine IP-Adresse wie `100.x.x.x`
- Diese IP bleibt immer gleich – egal wo du bist

## ✅ Test
Öffne die Eingabeaufforderung (cmd) und tippe:
```
tailscale status
```
Du solltest dein Gerät in der Liste sehen.

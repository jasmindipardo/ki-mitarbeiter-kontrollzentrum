# Tailscale einrichten – Mac

## Option A: Mac App Store (empfohlen)
1. Öffne den **App Store**
2. Suche nach "Tailscale"
3. Klicke auf **Laden** (kostenlos)

## Option B: Direkt herunterladen
1. Gehe zu [tailscale.com/download](https://tailscale.com/download)
2. Klicke auf **Download for macOS**

## Schritt 2: Einloggen
1. Tailscale in der Menüleiste (oben rechts) öffnen
2. Klicke auf **Log in**
3. Browser öffnet sich → einloggen

## Schritt 3: Netzwerkerlaubnis
macOS fragt nach Erlaubnis für VPN-Verbindung → **Erlauben**

## ✅ Test
Terminal öffnen und eingeben:
```
tailscale status
```
